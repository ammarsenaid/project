# FachkundePilot — UI Design Direction

## 1) Visual Style
Premium, clean, modern SaaS aesthetic with educational clarity and German-market trust.

## 2) Design Principles
1. Clarity over decoration.
2. Consistency over novelty.
3. Serious but approachable tone.
4. Fast comprehension on mobile.
5. Feedback-rich interactions for confidence.

## 3) Color Palette Recommendation
- **Background**: Warm White / Soft Ivory (`#FAF8F4`)
- **Primary Text**: Deep Navy (`#0F172A`)
- **Primary Accent**: Amber/Gold (`#D4A017`)
- **Secondary Accent**: Soft Blue (`#3B82F6`)
- **Success**: Muted Green (`#15803D`)
- **Warning**: Warm Orange (`#C2410C`)
- **Error**: Accessible Red (`#B91C1C`)

## 4) Typography Recommendation
- Primary: Inter or Plus Jakarta Sans for UI readability.
- Optional display accent: a restrained serif for marketing headers only.
- Scale:
  - H1/H2 bold and spacious.
  - Body text 16px baseline.
  - German terms emphasized with weight, not heavy color.

## 5) Layout Rules
- 12-column desktop grid, 4-column mobile behavior.
- Generous vertical rhythm (8px scale system).
- Max content width for reading-heavy pages.
- Persistent contextual side panel on large learning screens.

## 6) Component Style
- Rounded cards (12–16px radii).
- Subtle shadows (low blur, low opacity).
- Borders using soft neutrals.
- Icon-first indicators for progress/status.
- Minimal animation (150–250ms ease-out).

## 7) Dashboard Style
- Progress cards with clear numeric + visual indicators.
- Action-oriented hierarchy (Continue first).
- Avoid dense tables as default; prioritize cards and compact charts.

## 8) Card Style
- Certificate cards: title, status, progress ring, next action.
- Learning cards: concise metadata (duration, difficulty, completion).
- Flashcard shell: focused content area with distraction-free controls.

## 9) Mobile Style
- Sticky bottom primary navigation.
- Large tap targets (>=44px).
- Single primary CTA per screen.
- Minimize modal stacking; prefer full-screen sheets.

## 10) What to Avoid
- Childish illustrations and over-gamification.
- Heavy dark themes as default.
- Overly corporate grayscale monotony.
- Visual clutter and competing CTA buttons.
- Excessive motion and noisy gradients.
